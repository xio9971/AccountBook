## AccountBook (가계부)
#### 예산에 따른 지출내역 관리 어플리케이션

### 주요기능
- 한달 기준 예산입력
- 지출내역 추가및 관리
- 지출이 예산의 80%를 초과시 경고메세지 출력
- CoreData 사용
